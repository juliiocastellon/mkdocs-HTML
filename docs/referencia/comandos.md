# Comandos HTML

## Objetivo
Mostrar ejemplos básicos de uso de etiquetas HTML.

## Requisitos
- Tener conocimientos básicos de HTML
- Editor de código (VS Code)

## Pasos

### Crear un título
```html
<h1>Este es un título</h1>
```

### Crear un párrafo
```html
<p>Esto es un párrafo de texto</p>
```

### Crear un enlace
```html
<a href="https://www.google.com">Ir a Google</a>
```

### Insertar una imagen
```html
<img src="imagen.png" alt="Imagen de ejemplo">
```

### Crear una lista
```html
<ul>
  <li>Elemento 1</li>
  <li>Elemento 2</li>
</ul>
```

## Verificación
- Los elementos se muestran correctamente en el navegador
- No hay errores en el código

## Errores frecuentes
- No cerrar etiquetas
- Escribir mal las etiquetas
- Rutas incorrectas en imágenes o enlaces
