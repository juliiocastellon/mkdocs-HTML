# Quickstart

## Objetivo
Crear una página HTML básica desde cero y visualizarla en el navegador.

## Requisitos
- Tener un editor de código (VS Code)
- Tener un navegador web

## Pasos

### Crear archivo HTML
Crea un archivo llamado `index.html`

### Escribir estructura básica
```html
<!DOCTYPE html>
<html>
<head>
  <title>Mi primera web</title>
</head>
<body>
  <h1>Hola mundo</h1>
  <p>Esta es mi primera página web</p>
</body>
</html>
```

### Guardar el archivo
Guarda el archivo con extensión `.html`

### Abrir en el navegador
Haz doble clic en el archivo o ábrelo con el navegador

## Verificación
- Se muestra el texto "Hola mundo"
- El navegador interpreta correctamente el HTML

## Errores frecuentes
- No guardar el archivo como .html
- Olvidar etiquetas básicas como `<html>` o `<body>`
- Escribir mal las etiquetas
