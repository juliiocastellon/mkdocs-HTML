# Enlaces HTML

## Objetivo
Aprender a crear enlaces en HTML para navegar entre páginas web.

## Requisitos
- Tener un archivo .html
- Conocer la estructura básica de HTML

## Pasos

### Crear un enlace básico
```html
<a href="https://www.google.com">Ir a Google</a>
```

### Abrir enlaces en otra pestaña
```html
<a href="https://www.google.com" target="_blank">Abrir Google</a>
```

### Enlace a una página local
```html
<a href="pagina.html">Ir a otra página</a>
```

## Verificación
- Al hacer clic en el enlace, se abre la página correctamente
- El enlace funciona sin errores

## Errores frecuentes
- No poner el atributo href
- Escribir mal la URL
- Olvidar cerrar la etiqueta <a>
