# Imágenes en HTML

## Objetivo
Aprender a insertar imágenes en una página HTML.

## Requisitos
- Tener una imagen guardada
- Conocer la estructura básica de HTML

## Pasos

### Insertar una imagen básica
```html
<img src="imagen.png" alt="Descripción de la imagen">
```

### Imagen con tamaño
```html
<img src="imagen.png" alt="Imagen" width="300">
```

### Imagen desde internet
```html
<img src="https://via.placeholder.com/150" alt="Imagen de ejemplo">
```

## Verificación
- La imagen se muestra correctamente
- No aparece rota

## Errores frecuentes
- Ruta incorrecta de la imagen
- No poner el atributo alt
- Nombre de archivo mal escrito
